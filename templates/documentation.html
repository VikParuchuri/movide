Setting the theme style

Once you've made the hard choice of selecting which theme you'd like to use, there are few things to do to disable the style switching.

On each page update the <body> tag to have the correct class for your theme:

...
<body>
<!--
Use the corresponding body tag for your chosen theme
<body class="blue">
<body class="orange">
<body class="green">
<body class="bw">
-->
...

Remove the style switcher from the footer on all pages:

...
<div class="col-md-6">
	<p class="text-right">
	Style: <a href="" class="st" id="style-bw">Black & White</a>, <a href="" class="st" id="style-orange">Orange</a>, <a href="" class="st" id="style-green">Green</a>, <a href="" class="st" id="style-blue">Blue</a>
	</p>
</div>
...

In colorfrog.js comment out or delete the marked code blocks. There are four in total, below is an example of the first:

/*
* Set initial style
* REMOVE WHEN YOU HAVE PICKED YOUR STYLE
*/ 
...
// END OF CODE TO REMOVE

If you want to reduce the weight of the colorfrog.css file you can remove the style specific css that you dont need. Each block is marked as below:

/*
* Blue
*/
...
/*
* //end blue styles
*/

Likewise you can also delete the unused style image folders from the themes img folder.

Configuring Google maps

The footer map can easily be configured to your location and the pointer made to point exactly where you want. There are two pairs of coordinates that can be manipulated to change the map.

To center the map on your office/chosen place simply change the latitude and longitude of the following line in colorfrog.js:

...					
var mapOptions = {
	center: new google.maps.LatLng(60.170421,24.938149), 
...

To set the marker position simply change the latitude and longitude of the following line in colorfrog.js:

...
var myLatLng = new google.maps.LatLng(60.16992,24.938707);
...

Browser component

The browser component address bar can be figured in two different ways. You can choose to have it with a usable anchor tag or a abstract grey bar.

Use the following code for a browser with a usable anchor:

...					
<div class="addressbar">
	<div class="search text"><a href="http://leapfrogui.com">http://leapfrogui.com</a></div>
</div>
...

Use the following code for a browser with an abstract address:

...					
<div class="addressbar">
	<div class="search"></div>
</div>
...
